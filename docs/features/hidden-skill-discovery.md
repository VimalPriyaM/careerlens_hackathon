# Hidden Skill Discovery

Finds skills you have evidence for but forgot to put on your resume.

## How it works

During cross-referencing, the system identifies skills with status:
- **Hidden Skill:** Found on LinkedIn + GitHub, but missing from resume
- **Undiscovered:** Found on GitHub only, not on resume or LinkedIn

## What the user sees

Each hidden skill shows:
- The skill name
- Where it was found (GitHub, LinkedIn, or both)
- Action to take (e.g., "Add Docker to your resume — you have code evidence")

## Why it matters

Many developers forget to list skills they use daily. If your GitHub repos use Docker, CI/CD, or testing frameworks but your resume doesn't mention them, you're underselling yourself. These are quick wins — just add them to your resume.
