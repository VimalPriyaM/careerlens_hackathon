# Project Roadmap

LLM-generated project recommendations to close specific skill gaps.

## What each project includes

| Field | Description |
|-------|-------------|
| Title | What to build |
| Description | Detailed explanation |
| Tech stack | Technologies to use |
| Skills covered | Which target skills this addresses |
| Estimated hours | Time investment |
| Difficulty | Beginner / Intermediate / Advanced |
| GitHub push checklist | Specific files and steps to push |
| Evidence impact | Predicted score improvement (e.g., "Improves Node.js score by 25-30%") |

## How projects are generated

The LLM receives:
- All evidence scores (which skills are weak)
- Target role requirements
- User's current GitHub/LinkedIn/resume data

It generates projects ordered by impact — the first project covers the most critical gaps.

## Dashboard display

- Cards with accent bar (indigo for recommended first project)
- Numbered indicators
- Expandable GitHub checklist with progress tracking
- Evidence impact highlighted in green
