# Quick Wins

Low-effort actions for immediate score improvement.

## Examples

- "Add Docker to your resume skills section"
- "Request LinkedIn endorsements for Python"
- "Write a README for your api-project repo"
- "Add unit tests to your portfolio project"

## How they're generated

The LLM analyzes gaps between sources and suggests the easiest actions:
- Skills on GitHub but not resume → "Add to resume"
- Skills without endorsements → "Request endorsements"
- Repos without READMEs → "Write a README"

## Score Boost

When users check off quick wins on the dashboard:
- Each completed item adds to a score boost (up to +5 points)
- The boost is shown in the KPI cards and header
- This is client-side only — encourages the user to re-scan after completing actions

## Dashboard display

- Interactive checklist with progress bar
- Completion count (e.g., "3/5")
- "+N pts potential" indicator
- Celebration message when all completed
