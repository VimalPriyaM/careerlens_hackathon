# Prerequisites

## Software

| Requirement | Version | Download |
|-------------|---------|----------|
| Node.js | >= 18 | [nodejs.org](https://nodejs.org) |
| npm | >= 9 | Comes with Node.js |
| Git | Any | [git-scm.com](https://git-scm.com) |

## Accounts & API Keys

| Service | Purpose | Where to get it |
|---------|---------|----------------|
| **Supabase** | Database + Auth + Storage | [supabase.com](https://supabase.com) (free tier) |
| **Groq** | LLM API (Llama 3.3 70B) | [console.groq.com](https://console.groq.com) (free tier) |
| **GitHub PAT** | GitHub API access (5000 req/hr) | [github.com/settings/tokens](https://github.com/settings/tokens) (classic, no scopes needed) |
| **Vercel** | Frontend deployment | [vercel.com](https://vercel.com) (free tier) |
| **Railway** | Backend deployment | [railway.app](https://railway.app) (free trial) |
